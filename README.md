# 琢石寻光社会实践团网站

基于 Vue 3 + TypeScript + Vite 构建的社会实践团网站。

## 项目结构

```
src/
├── pages/          # 页面组件
├── router/         # 路由配置
├── components/     # 通用组件
├── assets/         # 静态资源
└── style.css       # 全局样式
```

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- Tailwind CSS
- Vue Router

## 运行项目

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 页面

- 首页 - 团队展示背景图
- 寻光团队 - 团队介绍
- 琢石纪事 - 实践活动记录
- 灵石文化 - 文化传承
- 山海传音 - 团队声音
